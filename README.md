# Mui_OnlineAppraise_SSM

基于Mui HTML5+plus的学生评教App应用

#### 目的：

通过该项目掌握SSM简单的数据增删查改和文件上传

#### 技术方案：

前端：Mui，H5+plus

后端：Spring+SpringMVC+MyBatis

#### 实现功能要点：

- 老师上传需评教的课程
- 学生对已上传的课程进行评价
- 学生评价成功后评分实时反馈显示
- 学生不能对同一课程重复评价
